## 介绍

由于 PHP 语言的特点, 或者说局限性, 他只能用于 web 开发, 所以他的框架不像其他语言的web框架
(node.js express/koa 或者 python flask/fastapi), 小而美PHP的框架一般都是大而全,
典型纯正的 MVC, 所有你 web 开发所需要用到的东西, 基本上都包括了

- 路由
- 配置
- 数据验证
- 模板引擎
- 数据库迁移
- 数据库orm
- 缓存
- 日志
- 调试
- 命令行
